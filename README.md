### `Run Locally:`

#### `Requirements:`

* Node.js (v14 or higher)  <img src="https://cdn.worldvectorlogo.com/logos/nodejs-icon.svg" width="25" height="25"/>

```bash
$ # change into react-ui directory
$ cd react-ui
$ # install dependencies
$ npm i
$ # start node.js server
$ npm start
```
