### `Run Locally:`

#### `Requirements:`

* Node.js <img src="https://cdn.worldvectorlogo.com/logos/nodejs-icon.svg" width="25" height="25"/>

```bash
$ # change into i-ching directory
$ cd i-ching
$
$ # install dependencies
$ npm i
$
$ # start node.js server
$ npm start
```
